## 制作について
### 制作期間
- 2026年2月17日

### こだわったポイント・学習したこと
- **Flexboxによるレイアウト**: `display: flex` を使用して、ロゴの中央寄せや詳細セクションの横並びを制御しました。
- **レスポンシブ対応**: メディアクエリを使用して、スマホ閲覧時に縦並びに切り替わるよう調整しました。
- **効率的な余白管理**: `margin: 0 auto` による中央寄せや、`max-width` を使ったコンテンツ幅の制限について学びました。

### 使用技術
- HTML5
- CSS3
